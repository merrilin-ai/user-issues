# Merrilin.ai User Issues

This repository is the public place for Merrilin.ai users to submit bug reports and product issues.

## Before opening an issue

- Search existing issues to avoid duplicates.
- Make sure you can reproduce the problem.
- Remove passwords, API keys, personal data, and other sensitive information from screenshots, logs, and descriptions.

## How to report a bug

1. Open a new issue using the **Bug report** template.
2. Describe what happened, what you expected to happen, and the steps needed to reproduce the issue.
3. Include your browser/device details and screenshots or logs when they are helpful and safe to share.

## What happens next

The Merrilin.ai team will review submitted reports, ask follow-up questions when needed, and use this repository to track status and resolution.

For urgent account, billing, or security concerns, please use the official Merrilin.ai support channels instead of posting publicly here.
